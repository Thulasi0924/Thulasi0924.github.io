Unilever Web Estate

Project Overview

The Unilever Web Estate project focused on optimizing the cloud infrastructure and content delivery networks (CDN) to enhance global web performance and security. The goal was to ensure faster website load times, improved security against cyber threats, and seamless global user experiences.

My Roles and Responsibilities

• Configured and optimized **Akamai Full CDN and Web Application Firewall (WAF)** to enhance website performance and security.

• Managed the migration of Unilever’s web infrastructure across **AWS, GCP, Azure, and various SaaS-based solutions.**

• Implemented **SSL, DNS, CORS, and URL redirection policies** to ensure web security and accessibility.

• Applied **security headers and firewall rules** to mitigate cyber threats and vulnerabilities.

• Assisted in troubleshooting **website downtime, latency issues, and performance bottlenecks.**

Technologies and Tools Used

Akamai | AWS | GCP | Azure | Terraform | Confluence | DNS Management Tools

Impact and Achievements

✅ Improved website load times and offloaded **60% of traffic from origin servers** using CDN optimizations.

✅ Enhanced security with **WAF rules and SSL management**, reducing security incidents.

✅ Successfully migrated web assets to a **multi-cloud environment**, ensuring high availability.
