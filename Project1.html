Unilever Q-Core Chatbot

Project Overview

The Unilever Q-Core Chatbot is an AI-powered chatbot designed to enhance customer interactions and automate responses. It integrates with cloud-based APIs and provides seamless communication for users.

My Role & Responsibilities

I was responsible for supporting the infrastructure of the Q-Core chatbot. My role included:

  • Ensuring high availability and performance of the chatbot infrastructure.
  • Monitoring system health, logs, and troubleshooting performance issues.
  • Optimizing cloud resources for cost-effectiveness and scalability.
  • Collaborating with development teams to ensure smooth deployment and updates.

Technologies & Tools Used

  • Akamai CDN & WAF for security and performance optimization.
  • Cloud services for hosting and scaling chatbot infrastructure.
  • Monitoring tools for real-time insights into chatbot performance.
  • CI/CD pipelines for seamless updates and maintenance.

Impact & Achievements

  • Ensured 99.9% uptime by proactively identifying and resolving infrastructure issues.
  • Reduced response latency by optimizing API integrations and caching mechanisms.
  • Strengthened chatbot security using web application firewall (WAF) configurations.
